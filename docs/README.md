# Sveltry documentation

Sveltry is an open-source, self-hosted, Sentry-compatible error tracker. Start
here.

## Get started

- [Quickstart](QUICKSTART.md) - send your first error in about five minutes.
- [Integrations](INTEGRATIONS.md) - install + initialize an SDK for your platform
  (browser, Node, Python, Go, Ruby, Java, PHP, Rust, frameworks).
- [Self-hosting](SELF_HOSTING.md) - run Sveltry in production: TLS, domains,
  storage, backups, and the environment-variable reference.

## Reference

- [Sentry compatibility](SENTRY_COMPATIBILITY.md) - which envelope item types,
  endpoints, and SDK features are supported, and the known limitations.
- [Public API](API.md) - the read/triage REST API authenticated by an org token.
- [Architecture](ARCHITECTURE.md) - how ingest, grouping, storage, and the live
  dashboard fit together.
- [Feature parity matrix](FEATURE_PARITY.md) - the honest current-vs-planned
  breakdown against Sentry.
- [Roadmap](ROADMAP.md) - what is next.
